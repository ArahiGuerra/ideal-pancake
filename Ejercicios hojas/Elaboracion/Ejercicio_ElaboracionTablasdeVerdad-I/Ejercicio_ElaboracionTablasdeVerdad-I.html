ELABORACIÓN DE TABLAS DE VERDAD (I)

Para el caso de una lógica bivalente (es decir, donde se consideran como posibles valores de cada variable proposicional sólo dos: verdadero y falso), responde usando números arábigos:

  • a. ¿Cuántas combinaciones de asignaciones de valores de verdad a cada una de las variables existen para un esquema proposicional con 2 variables (por ejemplo, p y q)?

  • b. ¿Cuántos renglones debe tener una tabla de verdad correspondiente a un esquema con 2 variables proposicionales?

  • c. ¿Cuántas combinaciones de asignaciones de valores de verdad a cada una de las variables existen para un esquema proposicional con 3 variables (por ejemplo, p, q y r)?

  • d. ¿Cuántos renglones debe tener una tabla de verdad correspondiente a un esquema con 3 variables proposicionales?

  • e. ¿Cuántas combinaciones de asignaciones de valores de verdad a cada una de las variables existen para un esquema proposicional con 5 variables (por ejemplo, p, q, r, s y t)?

  • f. ¿Cuántos renglones debe tener una tabla de verdad correspondiente a un esquema con 5 variables proposicionales?
